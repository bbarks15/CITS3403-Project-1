HTML

Hypertext Markup Language

History

Sir Tim Berners-Lee created HTML as an internet hypertext based system in late 1990 at CERN. HTML is a product of SGML (Standard Generalized Markup Language) which is a very complex, technical specification describing markup language. SGML was used to publish and exchange scientific and technical documents. HTML was created as an easier alternative to SGML. It would allow those who were not proficient in SGML to publish and exchange documents. One of the big features of HTML was the ability to hyperlink documents. This feature gave HTML its name, Hypertext Markup Language. Reference for information here. HTML was evolved greatly of the years. Outlined in the timeline below are the notable changes and information for each of the major HTML standards

Late 1990

HTML 1.0

  • The first version of HTML supported
    • Inline images
    • Text Controls
  • HTML 1.0 was very limited in terms of styling and presentation of content. For example you couldn't
    • Use tables, or frames,
    • Specify Fonts,
    • Change page background, or
    • Use forms
November 24, 1995

HTML 2.0

  • HTML 2.0 introduced the general HTML document structure. Using the <HTML>, <HEAD> and <BODY>
  • Introduced Several Document Type Definitions <DOCTYPE>
  • November 1995 Proposed formed based file upload
  • May 1996 Proposed markup for tables
  • August 1996 Proposed client-side image maps
  • January 1997 Added Support for Internationalization
January 14,1997

HTML 3.2

  • HTML 3.2 was published as a W3C recommendation.his was the first HTML specification published by W3C after taking over responsibility for the markup language.
  • Dropped math formulas entirely
  • Several features proposed as extensions to HTML 2.0 were incorporated
    • Standard markup for tables
    • Extension to the form feature to support uploading files
    • Support for client-side image maps
  • <DIV> Tag was added
  • Included "placeholder" elements <SCRIPT> and <STYLE>, intended to facilitate transition to the next version of HTML,
  • January 1997 Added Support for Internationalization
December 18, 1997

HTML 4.0

  • Extending the document character set from ISO-8859-1 to ISO 10646 (equivalent to Unicode)
  • Increased emphasis on use of external stylesheets via the <STYLE> element. Intended to phase out many of HTML's presentation elements and attributes.
  • Increased emphasis on the use of scripts and the <SCRIPT> element to allow authors to create dynamic Web pages and use HTML as a means to build networked applications.
  • <TABLE> The model for tables was extended to serve several objectives, including response to practical experience, support for accessibility, and to simplify import of tables.
  • The <OBJECT> element was introduced as a generalized approach for inserting multimedia
October 28, 2014

HTML 5.0

  • HTML 5 incorporated major changes and extensions over HTML 4.01.
  • Several elements were added for more explicit representation of document structure, including
    • <section>
    • <main>
    • <article>
    • <aside>
    • <header>
    • <footer>
    • <figure>
  • MathML and SVG markup can be used inside a document in the HTML serialization, using <math> and <svg> elements.
  • Flexible new elements were introduced for <audio> and <video> to avoid the need for format-specific plug-ins
  • Added <canvas> element to provided scripts with a rectangular area specified in pixels, to be used for rendering graphs, game graphics, or bit-mapped art on the fly.
  • Many detailed changes were introduced to provide more effective support for interactivity, for example, to support editing within web pages, with features such as drag-and-drop and spellchecking.

HTML has evolved from a simple way of sharing documents, to a intricate staple in modern web development. HTML provides the content of a webpage. Paired with the styling from CSS, and interactivity from Javascript it becomes a very powerful tool for creation.

Tags

HTML consists of tags denoted by the less-than < and greater-than > signs. These tags are interpreted and composed by the web browser to create text, images, and other material. Most tags consist of a start tag <TAG> and an end tag </TAG>. There are approximately 110 tags in the current HTML standard, HTML5. However, when HTML was first released, there were only 18 tags. Many of the current tags used today were not implemented in the original HTMl. A full list of the tags and the standard they were introduced can be found here.
